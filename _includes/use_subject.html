Use subject

Use subject instead of described_class.new when no argument is given to the initializer.

{% highlight ruby %} class Calculator attr_reader :base def initialize(base = 0) @base = base end def add(number1, number2) base + number1 + number2 end end {% endhighlight %}
{% highlight ruby %} describe '#add' do it 'sums two numbers' do calculator = described_class.new expect(calculator.add(1, 2)).to eq(3) end end {% endhighlight %}
{% highlight ruby %} describe '#add' do it 'sums two numbers' do expect(subject.add(1, 2)).to eq(3) end end {% endhighlight %}

It can be overridden when argument is given to the initializer.

{% highlight ruby %} describe '#add' do it 'sums two numbers' do subject = described_class.new(5) expect(subject.add(1, 2)).to eq(8) end end {% endhighlight %}
