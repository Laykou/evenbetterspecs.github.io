Factories, not fixtures

Factories are more flexible and easier to work with. Understand more here.

{% highlight ruby %} def full_name "#{first_name} #{last_name}" end {% endhighlight %}
{% highlight ruby %} it 'returns the full name' do user = create(:user, fist_name: 'Santos', last_name: 'Dumont') expect(user.full_name).to eq('Santos Dumont') end {% endhighlight %}

If your code doesn't hit the database, prefer build_stubbed over create since the former will make your tests faster ðŸ”¥

{% highlight ruby %} it 'returns the full name' do user = build_stubbed(:user, fist_name: 'Santos', last_name: 'Dumont') expect(user.full_name).to eq('Santos Dumont') end {% endhighlight %}
